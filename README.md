# showup-habit-tracker
ShowUp is a cross-platform Angular PWA habit tracker designed to make daily progress visible by tracking health and developer habits with a simple, offline-first, one-tap experience.

## Why this project?
The goal of ShowUp is simple: build consistency by showing up every day.  
Instead of focusing on outcomes, the app tracks daily actions to provide clear proof of progress over time.

## Features (MVP)
- Track daily habits (Health & Developer)
- One-tap mark as done
- Streak calculation
- Offline-first (PWA)
- Works on mobile and desktop

## Tech Stack
- Angular
- Angular PWA
- LocalStorage
